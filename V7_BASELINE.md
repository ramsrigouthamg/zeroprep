# ZeroPrep V7 baseline

V7 is the native Next.js deployment baseline.

- Presentation direction, motion, iconography, and exports are inherited from V6.
- PDF and PowerPoint exports are generated in the browser and downloaded locally.
- ZeroPrep does not upload, preserve, or expose presentation files through public URLs.
- Realtime and imagery routes include lightweight rate limits.
- The project builds with the native Next.js toolchain used by Vercel.
